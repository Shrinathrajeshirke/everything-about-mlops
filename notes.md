## 1. Envrironment preparation

### steps -
    1. create a github repository
    2. click on code -> codespaces -> create codespaces on main - new tab will open
    3. click on 3 lines -> open in VS code desktop
    4. install github codespaces and authorize github to vscode
    5. if anaconda not installed in machine --
    - go to base location: cd ..
    ```
    wget https://repo.anaconda.com/archive/Anaconda3-2022.05-Linux-x86_64.sh
    bash Anaconda3-2022.05-Linux-x86_64.sh
    ```
    6. check anaconda installed --
    open a new terminal
    
    check anaconda installed 
    ```
    which python
    ```

    check python version
    ```
    python -V 
    ```
